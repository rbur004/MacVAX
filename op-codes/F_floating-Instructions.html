F_floating Instructions

    This group of instructions may be omitted from the VAX instruction set. Attempted execution of an omitted Instruction will cause a reserved instruction fault. MacVAX implements this group.

    ACBF Add Compare and Branch Float
    ADDF2 Add Float (2 arg)
    ADDF3 Add Float (3 arg)
    CMPF Compare Float
    CVTBF Convert Byte to Float
    CVTFB Convert Float to Byte
    CVTFL Convert Float (truncated) to Long
    CVTFW Convert Float to Word
    CVTLF Convert Long to Float
    CVTRFL Convert Float (rounded) to Long
    CVTWF Convet Word to Float
    DIVF2 Divide Float (2 arg)
    DIVF3 Divide Float (3 arg)
    EMODF Extended Multiply and Integerize Double Float Not implemented
    MNEGF Move Negated Float
    MOVFMove Float
    MOVAF Move Address of Float
    MULF2 Multiply Float (2 arg)
    MULF3 Multiply Float (3 arg)
    POLYF Polynomial evaluation Float
    PUSHAF Push Address of Float onto Stack
    SUBF2 Subtract Float (2 arg)
    SUBF3 Subtract Float (3 arg)
    TSTF Test Float
