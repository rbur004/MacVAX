BPT Instruction

Purpose:Stop for debugging
Format:opcode
Operation:PSL<TP> <- 0;
{Break Point Fault}
Condition
Codes:		 N <- 0;
Z <- 0;
V <- 0;
C <- 0;
Exceptions:None
Opcodes:
03BPTBreak Point Fault
Description: This instruction is used, together with the T-Bit of the PSL, to implement debugging facilities.
