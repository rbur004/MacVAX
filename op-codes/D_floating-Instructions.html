D_floating Instructions

    This group of instructions may be omitted from the VAX instruction set. Attempted execution of an omitted Instruction will cause a reserved instruction fault. MacVAX Omits this group.

    ACBD Add Compare and Branch Double Float omitted
    ADDD2 Add Double Float (2 arg) omitted
    ADDD3 Add Double Float (3 arg) omitted
    CMPD Compare Double Float omitted
    CVTBD Convert Byte to Double Float omitted
    CVTDB Convert Double Float to Byte omitted
    CVTDF Convert Double Float to Float omitted
    CVTDL Conver Double Float (truncated) to Long omitted
    CVTDW Convert Double Float to Word omitted
    CVTFD Convert Float to Double Float omitted
    CVTLD Convert Long to Double Float omitted
    CVTRDL Convert Double Float (rounded) to Long omitted
    CVTWD Convert Word to Double Float omitted
    DIVD2 Divide Double Float (2 arg) omitted
    DIVD3 Divide Double Float (3 arg) omitted
    EMODD Extended Multiply and Integerize Double Float omitted
    MNEGD Move negated Double Float
    MOVDMove Double Float omitted
    MOVAD Move Address of Double MOVAQ
    MULD2 Multiply Double Float (2 arg) omitted
    MULD3 Multiply Double Float (3 arg) omitted
    POLYD Polynomial Evaluation Double Float omitted
    PUSHAD Push Address of Double onto Stack PUSHAQ
    SUBD2 Subtract Double Float (2 arg) omitted
    SUBD3 Subtract Double Float (3 arg) omitted
    TSTD Test Double Float omitted
