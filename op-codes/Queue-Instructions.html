Queue Instructions

    INSQUE Insert Into an Absolute Queue
    REMQUE Remove From an Absolute Queue

    MacVax has not implemented the following Queue instructions.

    INSQHI Insert Into a self-relative Queue Head (interlocked) not implemented
    INSQTI Insert Into a self-relative Queue Tail (interlocked) not implemented
    REMQHI Remove From a self-relative Queue Head (interlocked) not implemented
    REMQTI Remove From a self-relative Queue Head (interlocked) not implemented
