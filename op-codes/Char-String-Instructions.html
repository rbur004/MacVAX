Character String Instructions

    MOVC3 Move Characters (3 arg)
    MOVC5 Move Characters (5 arg)

    Optional

      Instructions in this group may be individually omitted from the VAX instruction set. Attempted execution of an omitted Instruction will cause a subset-emulation exception. MacVAX Omits most of this group.

      CMPC3 Campare Character (3 arg) omitted
      CMPC5 Campare Character (5 arg) omitted
      CRC Calculate Cyclic Redundancy Check omitted
      LOCC Locate Character
      MATCHC Match Character omitted
      MOVTC Move Translated Characters omitted
      MOVTUC Move Translated until Character omitted
      SCANC Scan for a Character omitted
      SKPC Skip Character
      SPANC Span Characters omitted
