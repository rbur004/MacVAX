Memory

MacVAX has 256K of Physical memory (this is not user alterable and may alter from version to version). A kernel mode process can determine the amount of physical memory by reading the internal processor register MVMEM. Physical addresses are considered 24 bit quantities, the top byte of each address being ignored. Virtual Memory is allocated in 512 byte pages. The mapping between physical and virtual addresses is done by the memory management hardware (see the Page Table Internal Processor Register).
